# bauzonenstatistik
Bauzonenstatistik 2.0

Ausscheidung von potentiell bebaubaren Flächen im Baugebiet (Bauzonen im Siedlungsgebiet) die breit und gross genug sind
um sinnvoll bebaut zu werden und noch nicht überbaut sind.

Weiters werden die Daten aggregiert und pro Zonentyp und Gemeinde ausgewiesen.

Schliesslich werden noch Parzellen ausgewiesen welche bebaubares Potenzial aufweisen.
